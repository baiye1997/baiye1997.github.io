# Hermes Agent 快速参考卡

---

## 📦 安装命令

```bash
# 一键安装
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash

# 重新加载shell
source ~/.bashrc

# 验证安装
hermes version
hermes doctor
```

---

## 🔧 配置命令

### 模型配置

```bash
# 交互式配置
hermes model

# 添加提供商
hermes model add <provider> \
  --base-url <url> \
  --api-key <key> \
  --model <model-id>

# 查看模型
hermes model list

# 测试模型
hermes model probe

# 切换模型
hermes config set model.default <model-id>
```

### 飞书配置

```bash
# 交互式配置（推荐）
hermes gateway setup

# 命令行配置
hermes config set channels.feishu.enabled true
hermes config set channels.feishu.app_id <app-id>
hermes config set channels.feishu.app_secret <secret>
hermes config set channels.feishu.connection_mode websocket
```

### YOLO模式

```bash
# 开启YOLO
hermes config set yolo.enabled true

# 关闭YOLO
hermes config set yolo.enabled false

# 查看状态
hermes config get yolo.enabled
```

## 🚀 服务管理

### Gateway控制

```bash
# 启动
hermes gateway start

# 停止
hermes gateway stop

# 重启
hermes gateway restart

# 查看状态
hermes gateway status

# 查看日志
hermes gateway logs

# 测试连接
hermes gateway probe feishu
```

### Systemd服务（开机自启）

```bash
# 启用
sudo systemctl enable hermes-gateway.service

# 启动
sudo systemctl start hermes-gateway.service

# 停止
sudo systemctl stop hermes-gateway.service

# 重启
sudo systemctl restart hermes-gateway.service

# 查看状态
systemctl status hermes-gateway.service

# 查看日志
journalctl -u hermes-gateway -f
```

## 🆘 故障排查

### 常见问题

```bash
# 1. 命令找不到
source ~/.bashrc

# 2. 查看错误日志
grep -i error ~/.hermes/logs/gateway.log

# 3. 查看连接状态
cat ~/.hermes/gateway_state.json

# 4. 测试飞书连接
hermes gateway probe feishu

# 5. 重启Gateway
pkill -f "hermes gateway"
hermes gateway start
```

### 日志查看

```bash
# 实时日志
tail -f ~/.hermes/logs/gateway.log

# 最近50行
tail -50 ~/.hermes/logs/gateway.log

# 错误日志
grep -i error ~/.hermes/logs/gateway.log

# Systemd日志
journalctl -u hermes-gateway -f
```

## ⚡ 快速测试流程

```bash
# 1. 安装
curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash && source ~/.bashrc

# 2. 验证
hermes version && hermes doctor

# 3. 配置模型
hermes config set OPENAI_API_KEY <key>
hermes config set OPENAI_BASE_URL <url>
hermes config set model.default mimo-v2-omni

# 4. 测试模型
hermes model probe

# 5. 配置飞书
hermes config set FEISHU_APP_ID <app-id>
hermes config set FEISHU_APP_SECRET <secret>
hermes config set channels.feishu.enabled true

# 6. 启动Gateway
hermes gateway start

# 7. 验证连接
hermes gateway status
```
