# 🎨 Obsidian 主题与插件推荐

> 来源：[awesome-obsidian](https://github.com/kmaasrud/awesome-obsidian)
> 挑选原则：简洁美观、实用易用、对日报系统有帮助

---

## 🎨 主题推荐（1个）

### Minimal
- **GitHub**: https://github.com/kepano/obsidian-minimal
- **简介**: 极简主义主题，专注于阅读体验，去除多余装饰
- **推荐理由**: 
  - Obsidian 社区最受欢迎的简洁主题
  - 配色柔和不刺眼，长时间阅读不累
  - 完美支持 Callout（[!tip] 等），我们日报的分类展示会更好看
  - 可配合 Style Settings 插件进行微调
  - 支持亮色/暗色模式切换

---

## 🔌 插件推荐（2个）

### 1. Templater
- **GitHub**: https://github.com/SilentVoid13/Templater
- **简介**: 强大的模板引擎，支持变量、脚本和自动化插入
- **推荐理由**:
  - 可创建每日报告模板，一键生成当天笔记
  - 支持动态内容（日期、天气、待办等）
  - 与 Dataview 配合使用效果更佳
  - 可简化日报的创建流程，减少重复劳动

### 2. Shell commands
- **GitHub**: https://github.com/Taitava/obsidian-shellcommands
- **简介**: 在 Obsidian 中预定义并运行系统命令，支持快捷键
- **推荐理由**:
  - 可在日报中一键执行同步命令（配合 obsidian-headless）
  - 支持自动执行，适合定时任务
  - 可调用外部脚本，扩展日报功能
  - 服务器环境特别实用

## 🎯 CSS Snippet 推荐（2个）

### 1. Stylish Block Quotes
- **GitHub 源文件**: [awesome-obsidian/code/css-snippets/stylish-blockquotes.css](https://github.com/kmaasrud/awesome-obsidian/blob/master/code/css-snippets/stylish-blockquotes.css)
- **简介**: 美化引用块和 Callout 的样式，增加视觉层次
- **推荐理由**:
  - 直接提升日报中 [!tip]、[!info] 等 Callout 的颜值
  - 无需配置，放入 snippets 文件夹即可生效
  - 与 Minimal 主题搭配效果很好

### 2. Hyphenation and Justification
- **GitHub 源文件**: [awesome-obsidian/code/css-snippets/hyphenation-and-justification.css](https://github.com/kmaasrud/awesome-obsidian/blob/master/code/css-snippets/hyphenation-and-justification.css)
- **简介**: 全文两端对齐 + 自动断词，改善排版效果
- **推荐理由**:
  - 让日报正文排列更整齐，阅读体验更像印刷品
  - 中文排版效果提升明显
  - 简单有效，不影响其他功能

## 📋 安装简要说明

1. **主题**: 设置 → 外观 → 社区主题 → 浏览 → 搜索 "Minimal" → 安装并启用
2. **插件**: 设置 → 第三方插件 → 关闭安全模式 → 浏览 → 搜索安装
3. **CSS Snippet**: 将 `.css` 文件放入 vault 的 `.obsidian/snippets/` 文件夹，然后在设置 → 外观 → CSS snippets 中启用
